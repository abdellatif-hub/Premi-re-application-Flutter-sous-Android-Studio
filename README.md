## Objectif :
Mon application Flutter contient une page d’accueil avec un titre "La page Home", un texte au centre et un bouton flottant.
Elle possède aussi un menu latéral (Drawer) avec une photo de profil et plusieurs boutons : Home, Contacts, ChatBot et Météo.
Chaque bouton permet de naviguer vers une autre page.
